# siyche.github.io

Personal site built with Jekyll and Hydejack.

## Local development

1. Install Xcode Command Line Tools with `xcode-select --install`.
2. Use Ruby `3.1.4`.
3. Run `bundle install` or `BUNDLE_PATH=vendor/bundle bundle install`.
4. Run `bundle exec jekyll serve` or `BUNDLE_PATH=vendor/bundle bundle exec jekyll serve`.
5. Open `http://127.0.0.1:4000`.

## Main files

- `_config.yml`: site metadata and theme settings
- `index.md`: homepage
- `about.md`: bio page
- `experience/index.md`: experience landing page
- `_posts/`: blog posts
